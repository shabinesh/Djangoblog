Simple Django Blog
==================

```
git clone <repo>
```

```
$virtualenv env
$. env/bin/activate
```

install `libjpeg-dev` for pillow to support jpeg files

```
pip install -r requirements.txt
```

```
./manage.py runserver
```

```
./manage.py test
```

Get Meteor Running as reactive frontend. 

```
git checkout meteor-frontend
$cd frontend
$meteor 
```
